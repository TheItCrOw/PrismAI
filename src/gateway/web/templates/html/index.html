PrismAI Gateway

A Toolbox for AI-generated Text Detection

PrismAI is an environment for the automatic detection of AI-generated content.

This includes a collection of datasets, trained models, and an interactive demo for users to test out the detection of AI-generated text. The goal of this project is to provide a comprehensive and user-friendly platform for researchers, developers, and anyone interested in the field of AI-generated content detection.

Everything within PrismAI is open-source.
Luminar

With PrismAI, we present Luminar, a state-of-the-art AI text detection model that leverages hidden likelihood spaces and CNNs to identify AI-generated content with high accuracy and tiny footprints. In this model family, we provide LuminarSeq as a free-to-use web demo for users to test out the detection of AI-generated text. LuminarSeq is designed to be fast and efficient, making it suitable for real-time applications and is based on sequential classification of AIGT, unlike other AIGT detectors.

PrismAI Dataset

In addition to Luminar, we also provide the PrismAI Dataset, a new dataset for training and evaluating AI text detection models. The PrismAI Dataset is a large-scale dataset that includes both human-written and AI-generated text, allowing for comprehensive evaluation of detection models. The data is gathered from a multitude of domains while using a variety of AI generators for the synthetization of the AI-generated text. The dataset is designed to be diverse and representative of real-world scenarios, making it suitable for training and evaluating AI text detection models. The PrismAI dataset covers the English and German languages.

Uniform Datasets

PrismAI contains a collection of datasets that have been standardized to a common format. These datasets include both human-written and AI-generated text, allowing for easy comparison and analysis. The datasets are sourced from various papers and prior efforts and cover a wide range of topics and styles. These datasets were projected to a common format to allow for easy usage and comparison.
